# 🎵 Music Player

A modern and responsive Music Player built using **HTML**, **CSS**, and **JavaScript**. It provides a clean UI with essential music playback controls such as Play/Pause, Previous, Next, Shuffle, Repeat, Mute, and a Progress Bar.

---

## 📸 Preview

> A simple and elegant music player interface with album artwork, track information, and playback controls.

---

## ✨ Features

- ▶️ Play / Pause music
- ⏮ Previous Track
- ⏭ Next Track
- 🔀 Shuffle Mode
- 🔁 Repeat Modes
  - Repeat All
  - Repeat One
  - Repeat Off
- 🔇 Mute / Unmute Audio
- 📈 Interactive Progress Bar
- ⏱ Current Time & Total Duration
- 🎨 Modern Responsive UI
- 🖼 Dynamic Album Cover Updates
- 🎵 Playlist Support

## 📁 Playlist

The playlist is defined inside `java.js`.

Each song contains:

```javascript
{
    title: "Song Title",
    artist: "Artist Name",
    cover: "Image URL",
    src: "Audio URL"
}
```

You can easily add more songs by inserting new objects into the `playlist` array.
